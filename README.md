📊 BlinkIT Sales Analysis Dashboard (Excel Project)
📌 Project Overview

This project is an interactive BlinkIT Sales Analysis Dashboard created in Microsoft Excel to analyze sales performance, outlet distribution, customer ratings, and product category trends.

The dashboard transforms raw retail sales data into meaningful business insights using charts, KPIs, slicers, pivot tables, and data visualization techniques.

The main objective of this project is to help stakeholders understand:

Sales performance over the years

Outlet-wise sales contribution

Product category performance

Customer buying patterns

Outlet size and location impact on sales.

📈 Dashboard Features
KPI Cards

The dashboard includes important business KPIs:

💰 Total Sales: 1.20M

📦 Total Items Sold: 8523

⭐ Average Rating: 3.97

📊 Average Sales: 140.99
